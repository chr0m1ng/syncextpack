Navitel Navigator-ի գործածության արտոնության պայմանագիր
Հարգելի՛ գնորդ,

Կարդացեք, խնդրեմ, հետևյալ Վերջնական Օգնատիրոջ արտոնագրային պայմանագիրը (հրապարակային առաջարկ)

ԱՐՏՈՆԱԳՐԱՅԻՆ ՊԱՅՄԱՆԱԳԻՐ «NAVITEL NAVIGATOR» ՆԱՎԻԳԱՑԻՈՆ ՀԱՄԱԿԱՐԳԻ ՕԳՏԱՏԻՐՈՋ ՀԵՏ

Սույն Արտոնագրային պայմանագիրն իրավաբանորեն պարտադիր պայմանագիր է, որ կնքվում է Ձեր (իբրև անհատի կամ իրավաբանական միավորի), այսուհետ՝ «Օգտատեր», և «Center of Navigation Technologies» փակ բաժնետիրական ընկերության միջև (այսուհետ՝ «NAVITEL ընկերություն»), ինչը վերաբերում է «Navitel Navigator» նավիգացիոն համակարգի Software ծրագրային ապահովման ապրանքատեսակի (այսուհետ՝ «Software ապրանքատեսակ»), գործածությանը։ Software ապրանքատեսակը ներառում է «Navitel Navigator» քարտեզագրական նավիգացիոն ծրագիր համակարգչային software (այսուհետ «Նավիգացիոն համակարգ»), թվայնացված քարտեզ (այսուհետ՝ «Թվային ապրանքատեսակ») և հարակից նյութեր՝ կամ տպագիր, կամ՝ էլեկտրոնային տարբերակով։ Թվային ապրանքատեսակը կարող է ձեռք բերվել. (i) Նավիգացիոն համակարգի հետ միասին, իբրև Software ապրանքատեսակի մաս, կամ (ii) իբրև նախկինում ձեռք բերված Software ապրանքատեսակի հետ համատեղ աշխատելու համար նախատեսված առանձին միավոր:

Սույն արտոնագրային պայմանագիրը տարածվում է ինչպես Software ապրանքատեսակի բաղադրամասերի թարմացումների, լրացումների, հավելյալ բաղադրիչների վրա, այնպես էլ Software ապրանքատեսակի՝ ապահովելով դրա գործունեությունը։ NAVITEL-ը Software ապրանքատեսակի բոլոր իրավունքների սեփականատերն է հետևյալ պայմաններով. քարտեզագրական բազաների և/կամ դրա առանձին տարրերի նկատմամբ բացառիկ հեղինակային իրավունքները տնօրինվում են դրանց սեփականատերերի կողմից՝ Software ապրանքատեսակի բոլոր օրինակների վրա դրված հեղինակային իրավունքների պաշտպանության նշանների համապատասխան։ Իրավունքներ ունեցողների ամբողջական ցանկի համար նայել ՀԵՂԻՆԱԿԱՅԻՆ ԻՐԱՎՈՒՆՔ հոդվածը։

NAVITEL-ն ունի Software ապրանքատեսակի տարածման բացառիկ իրավունք։
Software ապրանքատեսակն ամբողջությամբ և դրա տարրերն առանձին-առանձին (ներառյալ, սակայն ոչ սահմանափակված քարտեզագրական նյութերով, տվյալների բազաներով, գրաֆիկական պատկերներով, տեսագրություններով, ձայնագրություններով, տեքստով և լրացուցիչ ծրագրերով) հեղինակային իրավունքներ ունեն և պաշտպանված են Ռուսաստանի Դաշնության և միջազգային օրենսդրություններով։ Software ապրանքատեսակը տեղադրելով, կրկնօրինակելով, ներբեռնելով, հասանելիությունն իրականացնելով կամ Software ապրանքատեսակն այլ կերպ կիրառելով, Դուք այդպիսով համաձայնում եք սույն Արտոնագրային պայմանագրի դրույթների հետ։ Այս Վերջնական Օգտատիրոջ արտոնագրային պայմանագիրն ուժի մեջ է մտնում անմիջապես, երբ Դուք հաստատում եք Ձեր համաձայնությունը դրա հետ։ Այս Վերջնական Օգտատիրոջ արտոնագրային պայմանագիրն ուժի մեջ կլինի Software ապրանքատեսակի գործածության ողջ ընթացքում։ Եթե Դուք համաձայն չեք սույն Արտոնագրային պայմանագրի որևէ դրույթի հետ, մի՛ տեղադրեք և մի՛ գործածեք Software ապրանքատեսակը։

ՎԵՐՋՆԱԿԱՆ ՕԳՏԱՏԻՐՈՋ ԱՐՏՈՆԱԳՐԱՅԻՆ ՊԱՅՄԱՆԱԳԻՐ (ՀՐԱՊԱՐԱԿԱՅԻՆ ԱՌԱՋԱՐԿ)

Այս Software ապրանքատեսակը (ներառյալ տպագիր նյութերը) Ձեզ փոխանցվում է Արտոնագրային պայմանագրի դրույթների և պայմանների համաձայն և ՉԻ ՎԱՃԱՌՎՈՒՄ ՁԵԶ։


1. ՊԱՅՄԱՆԱԳՐԻ ԱՌԱՐԿԱՆ
1.1. NAVITEL-ը Ձեզ է տրամադրում մի պարզ, ոչ-բացառիկ արտոնագիր, այսինքն՝ Software ապրանքատեսակը գործածելու իրավունք՝ միայն ստորև ներկայացվող ձևերի և դրույթների համաձայն (այսուհետ՝ «Արտոնագիր»)։
1.2. Ստորև նախատեսված բոլոր դրույթները վերաբերում են Software ապրանքատեսակին ամբողջությամբ և դրա բոլոր տարրերին առանձին-առանձին։

2. ՄՏԱՎՈՐ ՍԵՓԱԿԱՆՈՒԹՅՈՒՆ
2.1. Software ապրանքատեսակի (ներառյալ, սակայն ոչ սահմանափակված ցանկացած տվյալների բազայի, գրաֆիկական պատկերների, լուսանկարների, տեսագրությունների, ձայնագրությունների և լրացուցիչ ծրագրերի իրավունքները, ինչպես նաև Software ապրանքատեսակի մեջ ընդգրկված այլ նյութերի նկատմամբ մտավոր սեփականության իրավունքները), հարակից տպագիր և էլեկտրոնային տեսքով նյութերի և Software ապրանքատեսակի բոլոր կրկնօրինակների սեփականատերը NAVITEL-ն ու դրա գործընկերներն են։ Սեփականության և հեղինակային բոլոր իրավունքները տվյալների բազաների և այլ բովանդակության նկատմամբ, որոնց հասանելիությունն է ապահովում Software ապրանքատեսակը, տնօրինվում են այդ տվյալների բազաների և առկա բովանդակության նկատմամբ սեփականության և հեղինակային իրավունքների սեփականատերերի կողմից և պաշտպանված են Ռուսաստանի Դաշնությունում գոյություն ունեցող օրենքներով և այլ ակտերով, ինչպես նաև մտավոր սեփականության դաշտում գործող Միջազգային իրավունքի ակտերով։ Սույն պայմանագիրը Ձեզ Software ապրանքատեսակի բովանդակության որևէ հասանելիություն չի շնորհում։
2.2. Software ապրանքատեսակի մասնավոր, ոչ-կոմերցիոն նպատակներով գործածելու պարզ, ոչ-բացառիկ արտոնագիրը տրվում է Օգտատիրոջն այնպես, ինչպես կանոնակարգված է պայմանագրով՝ այն դրույթները պահպանելու նկատառումով, որ սահմանված են ստորև։



2.3. NAVITEL-ը չի շնորհում որևէ այլ իրավունք, քան այն, ինչն ուղղակիորեն հայտարարագրված է ստորև։


3. ԳՈՐԾԱԾՈՒԹՅԱՆ ՊԱՅՄԱՆՆԵՐԸ
3.1. Software ապրանքատեսակը կարող է վաճառվել իբրև համակարգչի բաղադրամաս կամ համակցված լինել համակարգչին՝ նախօրոք դրանում տեղադրված վիճակում։ Եթե Օգտատերը ստանում է Software ապրանքատեսակն իբրև համակարգչի բաղադրամաս կամ եթե այն տրված է համակարգչի հետ համակցված, Software ապրանքատեսակն արտոնագրված է իբրև մեկ ինտեգրված ապրանք՝ միավորված համակարգչի հետ և կարող է գործածվել միայն այն համակարգչի հետ, որի կազմին է համակցված կամ որի փաթեթում որ այն առաքվել է։
Օգտատերը միշտ էլ կարող է փոխանցել սույն պայմանագրով իրեն տրված իրավունքներն այլ անձի, սակայն միայն համակարգչի համաժամանակյա և անվերադարձ վաճառքի և փոխանցման դեպքում, այն պարագայում, երբ Software ապրանքատեսակի ոչ մի բաղադրամաս կամ կրկնօրինակ Օգտատիրոջ մոտ չեն մնում և երբ Software ապրանքատեսակը ստացողը ստանձնում է սույն պայմանագրի դրույթներն ու պայմաններն իրականացնելու պարտավորությունը։
3.2. Software անպրանքատեսակը կարող է վաճառվել նույնիսկ առանց նախօրոք որևէ համակարգչի վրա տեղադրված լինելու և առանց որևէ համակարգչի հետ փաթեթի կազմում։ Եթե Օգտատերը ստանում է Software ապրանքատեսակը համակարգչից անջատ, Օգտատերն իրավունք ունի սույն պայմանագրով իրեն տրված իրավունքները Software ապրանքատեսակի բաղադրամասերի կամ կրկնօրինակների նկատմամբ փոխանցել այլ անձի այն պարագայում, եթե Software ապրանքատեսակի ոչ մի բաղադրամաս կամ կրկնօրինակ Օգտատիրոջ մոտ չեն մնում և եթե Software ապրանքատեսակը ստացողը պարտավորվում է ստանձնել և իրականացնել սույն պայմանագրի դրույթներն ու պայմանները։
3.3. Արտոնագրի շրջանակը։ Սույն պայմանագիրն Օգտատիրոջն իրավունք է շնորհում սարքի կոդի տեսքով վերարտադրելու Software ապրաքատեսակն համակարգչի հիշողության (RAM) մեջ այն ձայնագրելու միջոցով, ինչի նպատակը համակարգչի կենտրոնական պրոցեսորով Software ապրաքատեսակի ցուցումների ուղղակի կատարումն է։ Օգտատերն իրավունք չունի գործածել կամ պահպանել Software ապրանքատեսակն ավելի, քան մեկ համակարգչի վրա (միևնույնն է՝ տեղադրված, թե չտեղադրված տարբերակով)։
3.4. Software ապրանքատեսակում գտնվող Թվային ապրանքատեսակի գործածումը թույլատրվում է միայն Software ապրանքատեսակի հետ տրված Նավիգացիոն համակարգի հետ, և ոչ՝ այլ կերպ։ Թվային ապրանքատեսակը, որը ձեռք է բերվել իբրև առանձին միավոր, կարող է կիրառվել միայն մեկ Software ապրանքատեսակի մեջ (այսինքն՝ «հարմարեցվում է» մեկ եզակի Software ապրանքատեսակի Արտոնագրային բանալի)։
3.5. Օգտատերն իրավունք ունի Software ապրանքատեսակի կորած կամ վնասված օրինակը վերականգնելու համար խորհրդատվություն ստանալ՝ դիմելով NAVITEL-ի տեխնիկական աջակցության ծառայությանը։ Օգտատերն այդժամ պարտավոր է ներկայացնել Software ապրանքատեսակի իր օրինական գործածության վկայության կրկնօրինակը։
3.6. Տվյալների առաքում երկու տեսակի կրիչների վրա։ Այն դեպքում, եթե Software ապրանքատեսակն առաքվել է երկու կամ մի քանի տեսակի տվյալների կրիչների վրա, Օգտատերն իրավունք ունի գործածել միայն մեկը՝ իր համար ամենահարմար փաթեթը։ Ծրագրված է այնպես, որ Software ապրանքատեսակի մեկ օրինակի հետ առաքված բոլոր տվյալների կրիչները պարունակում են Software ապրանքատեսակի նույն օրինակը, ինչը կարող է Օգտատիրոջ կողմից գործածվել միայն մեկ համակարգչի վրա։
3.7. Օգտատերն իրավունք ունի կատարել Software ապրանքատեսակի միայն 1 (մեկ) կրկնօրինակ, որը պետք է գործածվի բացառապես պահուստային կրկնօրինակման (backup) նպատակով։
3.8. Օգտատերը պարտավոր է ինքը չվարվել և թույլ չտալ այլ անհատների կամ իրավաբանական միավորների վարվել հետևյալ կերպ.
- գործածել Software ապրանքատեսակն ուղղակի շահույթ հետապնդելու նպատակով,
- փոխառնել, վարձել կամ ժամանակավոր գործածության տրամադրել Software ապրանքատեսակը,
- Software ապրանքատեսակը գործածել առևտրային, ինչպես նաև ոչ-առևտրային նպատակներով համացանցային ծառայություններ մատուցելու համար, ներառյալ, բայց ոչ սահմանափակված համացանցային կայքէջերում հեռարձակումները,
- Software ապանքատեսակը գործածելու բարենպաստ պայմաններ ստեղծել այնպիսի մարդկանց համար, ովքեր անձնապես տվյալ Software ապրանքատեսակը գործածելու իրավունքներ չունեն (ներառյալ նրանք, ովքեր Ձեզ հետ աշխատում են միևնույն ցանցում կամ բազմակի օգտատերերի համակարգում), այլոց հատկացնել Software ապրանքատեսակի արտոնագիրը,
- տարածել (հրապարակել) Software ապրանքատեսակի ցանկացած հատվածներ կամ տարրեր, ներառյալ բայց ոչ սահմանափակված երրորդ կողմերին այն կրկնօրինակելու հնարավորություն ընձեռելը,
- փորձել շրջանցել Software ապրանքատեսակի ներքին տեխնիկական սահմանափակումները,
- ուղիղ մուտք գործել Թվային ապրանքատեսակի տվյալներ, հետազոտել Թվային ապրանքատեսակի կառուցվածքն ու պահուստի ձևաչափը, վերակազմավորել Թվային ապրանքատեսակի պահուստը, փոփոխել տվյալները, ներառյալ Software ապրանքատեսակի մեջ, փոփոխել (փոխարկել) և հարմարեցնել Software ապրանքատեսակը, ապաքանդել, կազմաքանդել (բնօրինակ տեքստում փոխել սարքի կոդը) ծրագիրը, տվյալները և Software ապրանքատեսակի այլ բաղադրիչները,
- սարքի ծրագրի կամ տվյալների կոդի փոփոխություններ անել, բացառությամբ այն դեպքերի, երբ դա կատարվում է այն միջոցներով, որ ներառված են Software ապրանքատեսակի փաթեթում և նկարագրված հարակից նյութերում,
- փոխանցել Software ապրանքատեսակի գործածության իրավունքները երրորդ կողմին, բացառությամբ այն դեպքերի, որոնք նկարագրված են սույն Արտոնագրային պայմանագրի 3.1. և 3.2 դրույթներում՝ ապահովելով սույն Արտոնագրային պայմանագրի պահանջների համապատասխանությունը որոշակիացված վերը նշված դրույթներում։


- Software ապրանքատեսակի նկատմամբ գործել խախտելով ինչպես Ռուսաստանի Դաշնությունում գործող օրենքներն ու ակտերը, այնպես էլ խախտելով միջազգային իրավունքի այն ակտերը, որոնք իրականացված են համակարգչային ծրագրային ապահովման ծրագրերի և տվյալների բազաների իրավական պաշտպանության դաշտում։
- գործածել Թվային ապրանքատեսակը որևէ այլ software ծրագրում, բացի Նավիգացիոն համակարգը։
3.9. Օգտատիրոջ կողմից 3.8. դրույթում բնորոշված գործողություններից որևէ մեկի կատարման դեպքում, «NAVITEL»-ն իրավունք է վերապահում միակողմանիորեն չեղյալ հայտարարել սույն Վերջնական Օգտատիրոջ արտոնագրային պայմանագիրն՝ առանց նախնական որևէ ծանուցման կամ նախազգուշացման։
3.10. Software ապրանքատեսակի գործածության հետ կապված միակ պատասխանատուն և ռիսկերի կրողն Օգտատերն է։
3.11. Software ապրանքատեսակի Արտոնագրային բանալու ակտիվացումն օրինական մեխանիզմ է, որ հաստատում է Օգտատիրոջ կողմից Software ապրանքատեսակի բարեխիղճ գործածությունը։
3.12. Software ապրանքատեսակի որևէ կրկնօրինակի Արտոնագրային բանալու ապաակտիվացում թույլ է տրվում ոչ ավելի, քան մեկ անգամ (ցանկացած պատճառով)։ Առանձին ձեռք բերված Թվային ապրանքատեսակի Արտոնագրային բանալու ապաակտիվացում գոյություն չունի։ Արտոնագրային բանալու վերաակտիվացում թույլ է տրվում միայն նույն մոդելի պաշտոնապես նույն օպերացիոն համակարգի վրա։ Արտոնագրային բանալու ապաակտիվացման/վերաակտիվացման մասին առավել մանրամասն տեղեկությունների համար նայեք «NAVITEL»-ի պաշտոնական http://navitel.cz կայքէջը։

3.13. ԽՆԴՐԵՄ ՀԱՇՎԻ ԱՌԵՔ, որ «NAVITEL»-ը տեխնիկական աջակցություն է ցուցաբերում միայն Software ապրանքատեսակի պաշտոնական տարբերակին և վերջինիս «NAVITEL»-ի կողմից կատարված թարմացումներին՝ գնումը կատարելուց հետո 1 (մեկ) տարվա ընթացքում։

Այլ ընկերությունների կամ/և անհատների կողմից մշակված/փոփոխված Նավագացիոն համակարգերի տարբերակների, թվայնացված ապրանքատեսակների և թարմացումների գործածությունը կարող է անկիրառելի դարձնել Software ապրանքատեսակն ամբողջությամբ կամ դրա առանձին տարրերը, ինչպես նաև կարող է վնաս հասցնել Սարքին՝ այն նորոգելու պահանջ առաջացնելով։ Օգտատերն իր սեփական ռիսկով և պատասխանատվությամբ է կարգավորում նման զարգացումները։


4. ԹԱՐՄԱՑՈՒՄ
4.1. NAVITEL-ի համաձայն թարմացման կատեգորիայի մեջ ընկնող software-ը գործածելու համար, անհրաժեշտ է ունենալ Software ապրանքատեսակի արտոնագիր, որն ընկնում է թարմացումների NAVITEL-ի առաջարկի տակ։ Թարմացումը մասնակիորեն կամ ամբողջությամբ փոխարինում կամ ընդլայնում է Software ապրանքատեսակը՝ այսպիսով դառնալով թարմացում ստանալու Ձեր իրավունքի վկայությունը։ Ձեր համակարգչի վրա թարմացումը տեղադրելիս, Դուք իրավունք ունեք գործածել թարմացումը միայն սույն պայմանագրի դրույթների ու պայմանների համաձայն, կամ՝ համաձայն թարմացմանը հարակից Արտոնագրային պայմանագրի դրույթների։
